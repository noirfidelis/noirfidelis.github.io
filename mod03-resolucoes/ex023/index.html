  7. Devido a grandes quantidades de dados em uma tabela, para não nos perdermos, podemos fixar o título usando:

  8. Podemos mesclar colunas e linhas em uma tabela. Para isso, utilizamos, respectivamente:
    1. col / row
    2. rowspan / colspan
    3. colspan / rowspan ✔
    4. rowspan / col
  9. Para colocar títulos em uma tabela, usamos a propriedade:
    1. <caption> ✔
    2. <title>
    3. <capture>
    4. <h1>
  10. Sempre que colocarmos "th" em nossa tabela, o W3C recomenda que utilizemos scopos para dar semântica ao nosso código HTML. Agora, quando queremos abranger mais de uma "linha" ou "coluna" em nossa tabela, usamos, respectivamente, os parâmetros:
    1. colgroup / rowgroup
    2. row / col
    3. colgroup / row
    4. rowgroup / colgroup ✔
