  1. A sigla CSS significa:
    1. Content Style Setup.
    2. Content Smart Sheets.
    3. Cascading Style Sheets. ✔
    4. Cascading Setup Style.
  2. Ao configurar os estilos com a técnica inline, as propriedades CSS são colocadas em que local?
    1. Dentro da própria tag HTML, em um parâmetro chamado style. ✔
    2. Dentro da área <head> do documento HTML, na área delimitada por <style>.
    3. Em um arquivo separado dentro da própria pasta onde o documento HTML está.
    4. Dentro de uma pasta especial chamada /style que fica no diretório atual do site.
  3. O que aconteceu com configurações como <body bgcolor="blue"> e <font-color="yellow"> que eram usadas em sites mais antigos?
    1. Continuam funcionando perfeitamente e podem ser usadas normalmente.
    2. Continuam funcionando, mas não é nada recomendável, pois as CSS são mais atuais. ✔
    3. Não funcionam mais de forma alguma.
    4. Ainda são a única opção de mudar cores.
  4. Qual das opções a seguir é a única que configura corretamente a cor da letra de um título usando CSS inline?
    1. <h1 color="blue">Título</h1>
    2. <h1 style="blue">Título</h1>
    3. <h1 style="font-color: blue">Título</h1>
    4. <h1 style="color: blue">Título</h1> ✔
  5. Para usar a técnica de estilos internos, a tag <style> deve estar em que local?
    1. Logo depois da tag HTML que queremos configurar.
    2. Dentro da área <head> do seu documento HTML. ✔
    3. No final do arquivo, logo depois do </body>
    4. Em qualquer lugar do arquivo HTML.
  6. Em um mesmo documento HTML, utilizamos as técnicas de CSS interno e CSS inline para mudar a cor de um elemento e acabamos usando duas cores diferentes. Quais configurações vão prevalecer na nersão final do site?
    1. As configurações duplicadas feitas nas CSS inline vão prevalecer.
    2. As configurações duplicadas feitas nas CSS internas vão prevalecer. ✔
    3. As duas configurações vão prevalecer e o texto será exibido duplicado com cores diferentes.
    4. O seu navegador vai decidir qual cor utilizar. Não existe uma regra para esses casos.
  7. Para utilizar estilos CSS externos, devemos usar a tag <link> no documento HTML para fazer o carregamento.
    1. <style>
    2. <css>
    3. <external>
    4. <link> ✔
  8. A tag de carga de estilos externos deve estar localizada em que local do documento HTML?
    1. Deve ser a primeira linha do arquivo HTML.
    2. Deve estar dentro da área <body> do documento.
    3. Deve estar dentro da área <head> do documento. ✔
    4. Pode estar em qualquer lugar, desde que esteja dentro do documento HTML.
  9. Para que os arquivos externos em CSS tenham compatibilidade com caracteres acentuados, devemos adicionar a regra:
    1. @UTF-8;
    2. @charset"UTF-8"; ✔
    3. @charset"PT-BR";
    4. @PT-BR;
  10. No fim das contas, vamos dar preferência à técnica de CSS externo sempre que for possível e vamos tentar evitar a técnica de CSS inline em nossos sites.
    1. CSS externo / CSS interno.
    2. CSS inlie / CSS interno.
    3. CSS externo / CSS inline. ✔
    4. CSS interno / CSS externo.
