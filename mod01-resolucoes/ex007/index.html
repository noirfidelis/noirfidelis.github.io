  1. Normalmente, nossos textos são organizados em sessões, e cada uma pode possuir um título. Em HTML5, existe alguma limitação em relação a quantidade de títulos que um documento pode ter?
    1. Sim. Só podemos ter no máximo seis títulos.
    2. Não. Um texto pode ter quantos títulos e quantos níveis for necessário.
    3. Não. Um texto pode ter vários títulos, mas apenas um máximo de seis níveis. ✔
    4. Sim. Podemos ter até doze títulos, dois para cada nível.
  2. Os títulos podem ser criados em seis níveis diferentes, usando as tags:
    1. <head 1> até <head 6>
    2. <header 1> até <header 6>
    3. <title 1> até <title 6>
    4. <h1> até <h6> ✔
  3. Se estamos em um ponto do texto com título nível 3 e queremos aprofundar esse assunto com um sub-título, devemos usar qual nível? Lembre-se de seguir a recomendação da W3C
    1. Nível 4. ✔
    2. Nível 2.
    3. Nível 3.
    4. Qualquer nível, de 3 até 6.
  4. O título principal de um determinado documento ou página deverá sempre ser definido com um título de qual nível?
    1. Nível 1. ✔
    2. Nível 2.
    3. Nível 3.
    4. Tanto faz, o que importa é ser um título.
  5. Muitos dizem que um documento só deve ter um título de nível 1 (principal). Essa crença é uma...
    1. Verdade. Somente um título deve ser o principal e todos os demais deverão ter outros níveis.
    2. Lenda. A recomendação é que não exista nenhum título de nível 1.
    3. Verdade. Se colocarmos mais de um título nível 1, seu site não será indexado pelo Google.
    4. Lenda. É possível adicionar mais de um título de nível 1, só devemos nos dedicar a descrevê-lo muito bem para facilitar a indexação por mecanismos de busca. ✔
  6. O estudo da indexação do conteúdo de um site por meio de mecanismos de busca é conhecido popularmente como:
    1. SEO - Seek Energy Organization.
    2. SEO - Search Engine Optimization. ✔
    3. SEO - Seek Engine Order.
    4. SEO - Search Elements Order.
  7. Todo documento HTML deverá ter obrigatoriamente algum nível 6?
    1. Não. Os títulos de menor nível serão usados apenas se for necessário. ✔
    2. Sim. Pelo menos um título de nível 6.
    3. Não. Os títulos de nível 6 devem ser evitados para não confundir o visitante.
    4. Sim. É necessário ter dois ou mais títulos de nível 6.
  8. Quando estamos desenvolvendo um site e o conteúdo ainda não foi definido, uma técnica bastante utilizada é usar blocos de texto genéricos com conteúdo composto por palavras em latim. Esses blocos são mais conhecidos pelo termo:
