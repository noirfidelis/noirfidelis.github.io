  1. Os símbolos de < e de > são usados em HTML para delimitar tags. Mas se quisermos mostrá-los no navegador, precisamos representá-los como:
    1. & l t ; e & g t ; ✔
    2. #lt; e #gt;
    3. &menor; e &maior
    4. #<; e #>;
  2. Dentro de um parágrafo no código HTML, se escrevermos uma palavra em cada linha, elas serão apresentadas no navegador de que maneira?
    1. Uma em cima da outra.
    2. Uma ao lado da outra. ✔
    3. Exatamente da maneira que digitamos.
    4. Nenhuma palavra será exibida.
  3. A qualquer momento do código HTML, se quisermos criar uma quebra de linha, devemos usar qual tag?
    1. <row>
    2. <new>
    3. <br> ✔
    4. Não precisa usar tag, é só apertar ENTER
  4. A melhor estratégia para criar um espaço relativo a dez quebras de linha em nosso site é:
    1. Usar a tag de quebra (br) por 10 vezes consecutivas.
    2. Usar a tag de parágrafo por 5 vezes consecutivas.
    3. Usar configurações de estilo em CSS e especificar o espaçamento todo de uma vez. ✔
    4. Não existe a possibilidade de criar um espaçamento desse tamanho.
  5. Os códigos como & euro; para mostrar o símbolo € nas páginas são conhecidos pelo nome:
    1. HTML entities. ✔
    2. Symbol codes.
    3. HTML symbols.
    4. Symbols entities.
  6. Que código utilizaremos para mostrar uma seta apontando para a direita (→) na página?
    1. &right;
    2. &rarrow;
    3. &arrow;
    4. & rarr; ✔
  7. Qual site é recomendado nas aulas para buscar os códigos usados para representar os emojis?
    1. WikiMoji
    2. EmojiBook
    3. EmojiPedia ✔
    4. SymbolPedia
  8. Se o código de um emoji é U + 1 F913, como devemos representá-lo em nosso código HTML?
    1. U + 1F913;
    2. &#X + 1F913;
    3. &#XU1F913;
    4. & #X1F913; ✔
  9. A marca br significa:
    1. Que o idioma está em português do Brasil.
    2. Break Row, ou quebra de linha. ✔
    3. Break Right, ou quebra à direita.
    4. Breaked Row, ou linha quebrada.
