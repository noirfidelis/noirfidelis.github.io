  1. Quando você está criando um site ou qualquer outro tipo de conteúdo, deve se preocupar bastante com os direitos de uso de determinada mídia (imagens, som, vídeo etc.). Qual dos itens a seguir melhor define esse assunto?
    1. Se a mídia estiver em um site público como Google e Youtube, podemos usar sem problemas.
    2. Não podemos usar nenhuma mídia sem a prévia autorização do seu criador. ✔
    3. Enquanto estamos estudando e aprendendo, temos total permissão para usar mídias de terceiros. ✔
    4. Podemos usar qualquer mídia e pagar uma multa quando formos descobertos.
  2. Qual dos itens a seguir é incorreto quando o assunto é não passar por problemas relacionados a direitos autorais?
    1. Seja o criador das suas próprias artes.
    2. Consiga uma autorização legal do detentor dos direitos autorais.
    3. Compre os direitos de uso em sites especializados e use quantas vezes quiser. ✔
    4. Use imagens com licenças públicas, como Creative Commons.
  3. Qual dos sites a seguir é especializado em disponibilizar apenas imagens para uso liberado, sem direitos autorais?
    1. Google Imagens.
    2. Getty Imagens.
    3. Shutterstock.
    4. Pexels. ✔
  4. Qual foi o formato de imagens criado na década de 80 com o objetivo de reduzir o tamanho dos arquivos das imagens, sem reduzir sua dimensão?

    4. ZIP

  7. Muita gente fica na dúvida sobre o tamanho da imagem:
    1. Use sempre a melhor qualidade possível para que seu site fique bonito.
    2. Use sempre a pior qualidade possível, pois os sites precisam ser rápidos.
    3. Tente balancear sempre qualidade e tamanho, para que seu site não fique pesado, mas não se esqueça da experiência proporcionada ao seu visitante. ✔
    4. Deixe o visitante escolher se quer imagens grandes ou sites rápidos.
