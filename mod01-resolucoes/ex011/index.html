  1. Qual das informações a seguir é a mais correta em relação ao cuidado que devemos ter em relação ao uso de imagens em sites?
    1. Use sempre a imagem com maior qualidade e resolução possível.
    2. Use sempre a imagem com menor qualidade e resolução possível.
    3. Use sempre a imagem com qualidade e resolução balanceadas. ✔
    4. Use sempre a menor quantidade de imagens para não deixar o site pesado.
  2. Deixar um site "pesado" com imagens grandes pode gerar:
    1. Aumento nas visitas por conta da qualidade visual.
    2. Diminuição na taxa de retenção e possível penalização em mecanismos de busca. ✔
    3. Aumento na taxa de retenção, pois o usuário passa mais tempo no site.
    4. Diminuição no tempo das visitas, mas isso acaba melhorando a indexação.
  3. Para usar imagens dinâmicas em nossos sites, devemos aprender a usar três tags, que são:
    1. <picture>, <source> e <img> ✔
    2. <image>, <picture> e <source>
    3. <picture>, <image> e <img>
    4. <img>, <source> e <image>
  4. A tag <source> possui o parâmetro media para configurar as dimensões da mídia, o type para indicar o formato da mídia e o parâmetro srcset para indicar o arquivo do destino. A única opção que mostra os parâmetros que preenchem as colunas na ordem correta é:
    1. media, type e srcset. ✔
    2. type, media e srcset.
    3. size, format e src.
    4. size, type e src.
  5. Podemos adicionar arquivos de áudio reproduzíveis às nossas páginas usando a tag:
    1. <sound>
    2. <music>
    3. <podcast>
    4. <audio> ✔
  6. Qual dos valores a seguir é o único que não é um parâmetro da tag de áudio?
    1. controls
    2. autoplay
    3. player ✔
    4. loop
  7. Quando inserimos um áudio no site, qual é o comportamento padrão do navegador em relação à carga do conteúdo?
    1. O arquivo de áudio não será carregado até que o visitante aperte o play. ✔
    2. O arquivo de áudio será completamente carregado, mesmo que nunca seja apertado o play.
    3. Os primeiros minutos de áudio serão carregados para facilitar a reprodução.
    4. Apenas os metadados do arquivo (informações básicas sobre o arquivo) serão carregados automaticamente.
  8. Para carregar vídeos hospedados localmente, podemos usar a tag HTML:
    1. <video> ✔
    2. <iframe>
    3. <media>
    4. <youtube>
  9. Quais são os formatos de vídeo que devemos considerar para manter a compatibilidade com a maioria dos navegadores?
    1. .mov .mp4 .avi .webm
    2. .m4v .ogv .avi .mov
    3. .mp4 .m4v .webm .ogv ✔
    4. .mp4 .mov .webm .avi
  10. Para inserir vídeos do Youtube no nosso site, usamos a tag:
    1. <video>
    2. <iframe> ✔
    3. <link>
    4. <media>
